### **[PitchFlows](https://pitchflows.online)** — AI Cold Email SaaS
*FastAPI · React · PostgreSQL · Neon · Claude Haiku/Sonnet*

**What it does:** AI-powered cold email outreach platform with intelligent reply automation.

**Key features:**

### **[Trading Bot](https://alfred-khaki.vercel.app)** — ML-Driven Algorithmic Trading
*LightGBM · FastAPI · Alpaca API · React/Vite · PostgreSQL*

**What it does:** Full-stack algorithmic trading system with hybrid signal generation and risk management.

**Engineering highlights:**
- **39-feature ML pipeline:** Price action, momentum, volatility, volume, structural, macro, sentiment
- **4-signal hybrid engine:** Technical (SMA stack + Chaikin Money Flow + Keltner Channels) + Fundamental + Sentiment + LightGBM alpha
- **Risk framework:** 95% VaR guard, ATR-scaled stops, regime-specific position sizing, Half-Kelly allocation
- **Backtesting:** TimeSeriesSplit validation, 15+ year historical data, drawdown circuit breaker
- **UI:** Finance terminal aesthetic (3-column grid, Outfit + JetBrains Mono, dark amber/teal)
- **Real-time signals** with WebSocket integration

**Status:** Paper trading with live signal generation | [Dashboard](https://alfred-khaki.vercel.app)
